# 📊 Data Analysis on Fortune 500 Companies

An Exploratory Data Analysis (EDA) project that extracts insights from Fortune 500 company data using Python, Pandas, and data visualization libraries.

---

## 📌 Project Overview

Fortune 500 is an annual list of the 500 largest US companies ranked by revenue. This project scrapes, cleans, and analyzes this data to uncover patterns in industry performance, revenue distribution, and employee satisfaction.

---

## 🎯 Objectives

- Extract company data using web scraping techniques
- Clean and preprocess raw data using Pandas
- Perform exploratory data analysis (EDA)
- Build visualizations to identify key business insights
- Support data-driven decision-making for corporate analysis

---

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| Python | Core programming language |
| Pandas | Data loading, cleaning, manipulation |
| NumPy | Numerical computations |
| Matplotlib | Static visualizations |
| Seaborn | Statistical visualizations |
| Requests + BeautifulSoup | Web scraping |

---

## 🔍 Key Insights Discovered

- Top industries by total revenue and company count
- Revenue distribution across sectors
- Employee count vs. revenue correlation
- Profit margin analysis by industry
- Year-over-year ranking changes
